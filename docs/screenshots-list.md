## Capture Tips

1. **Browser:** Use Chrome DevTools → Device Toolbar → set to 1440×900
2. **Theme:** Ensure dark mode is active (default)
3. **Data:** Run demo pipeline first for populated charts
4. **Format:** PNG, 2x resolution for Retina
5. **Annotations:** Use Figma or Excalidraw for callout overlays
